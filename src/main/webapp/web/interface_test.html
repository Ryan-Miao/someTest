接口就是一个服务请求，接口是一个网址，比如百度搜索jquery：
你会看到地址栏是这样的：http://www.baidu.com/s?wd=jquery&ie=UTF-8
其中，http://www.baidu.com/这是域名，s是这个域名下的某个路径，?后面是参数。这个地址的意思是：
我请求baidu的s路径，传过去的参数为wd，ie。参数值分别为：wd=jquery,ie=utf-8.
写代码要主要编码格式，说多了也不懂，你只需要知道统一成utf-8就可以了。
所以接口的意思很容易理解，?号之前的是请求的路径，后面的是参数。
那么请求完接口会返回什么呢，我们看到浏览器跳转到一个查询结果的页面，实际上这个就是返回的东西。
